### Manual Setup (If Preferred)
1. Create virtual environment: `python -m venv venv`
2. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Linux/Mac)
3. Install: `pip install -r requirements.txt`
4. Migrate: `python manage.py migrate`
5. Create admin: `python manage.py createsuperuser`
6. Run: `python manage.py runserver`

## ⚙️ Default Configuration

```python
DEBUG = True                              # Development mode
ALLOWED_HOSTS = ['*']                    # Allow all hosts
DATABASE = SQLite3                       # Development DB
FACE_RECOGNITION_TOLERANCE = 0.6         # Match threshold
CLASS_KEY_VALIDITY_HOURS = 8             # Key expiry
SESSION_TIMEOUT = 86400                  # 24 hours
```

### For Production, Change To:
```python
DEBUG = False
ALLOWED_HOSTS = ['yourdomain.com']
DATABASE = PostgreSQL
SECURE_SSL_REDIRECT = True
SECRET_KEY = 'random-secret-key'
```

## 🔐 Important Security Notes

⚠️ **Development Only:**
- Default settings are for development
- DEBUG is enabled
- SQLite is used (single user)
- ALLOWED_HOSTS set to '*'

⚠️ **Before Production:**
- Change SECRET_KEY
- Set DEBUG = False
- Use PostgreSQL
- Enable HTTPS
- Update ALLOWED_HOSTS
- Set secure cookies
- Configure backups
