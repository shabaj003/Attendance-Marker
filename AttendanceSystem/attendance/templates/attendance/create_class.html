About Class Keys

What is a class key?

A unique 6-character code generated automatically for each class session. Students use this code to mark their attendance.

Key Features:

  • ✓ Automatically generated (unique for each class)
  • ✓ Valid only during the class session period
  • ✓ Can be regenerated if needed
  • ✓ Prevents unauthorized attendance marking
